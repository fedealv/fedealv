3. IP

Que es?

Una dirección IP (Protocolo de Internet) es un número que identifica de manera única un dispositivo en una red que utiliza el protocolo de Internet para comunicarse. Permite que los dispositivos se comuniquen entre sí y facilita el enrutamiento de datos a través de Internet.

Tipos

Direcciones IP públicas: Son direcciones únicas utilizadas en Internet que pueden ser accesibles desde cualquier parte del mundo. Identifican dispositivos directamente en la red global de Internet.
Direcciones IP privadas: Son direcciones que se utilizan en redes locales y no son accesibles directamente desde Internet. Estas direcciones se utilizan para identificar dispositivos dentro de una red privada y suelen comenzar con ciertos rangos de direcciones designados para uso privado.

Clases

Las direcciones IP se dividen en diferentes clases que determinan el rango de direcciones disponibles para redes de diferentes tamaños:

Clase A: Se utiliza para redes grandes, ya que permite una gran cantidad de hosts, pero tiene menos redes disponibles.
Clase B: Se utiliza para redes de tamaño mediano, con un equilibrio entre redes y hosts.
Clase C: Se utiliza para redes pequeñas, ya que permite una gran cantidad de redes pero solo unos pocos hosts por red.

Direccionamiento

El direccionamiento IP se refiere al proceso de asignar direcciones IP únicas a dispositivos en una red. Las direcciones IP se asignan para permitir la comunicación y el enrutamiento eficientes de datos en una red.

Subnetting

El subnetting es el proceso de subdividir una red IP única en subredes más pequeñas para mejorar el rendimiento y la eficiencia de la red. Permite la gestión más eficiente de direcciones IP y ayuda a reducir el tráfico de red al dividir una red en segmentos más manejables.

VLSM

VLSM es una técnica de subnetting que permite a los administradores de redes subdividir una red IP en subredes de diferentes tamaños, lo que proporciona una mayor flexibilidad en la asignación de direcciones IP. Con VLSM, se pueden asignar más direcciones a subredes más grandes y menos direcciones a subredes más pequeñas, lo que optimiza el uso de direcciones IP disponibles en una red.
