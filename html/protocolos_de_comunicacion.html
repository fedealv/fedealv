Protocolos de Comunicacion

4. Protocolos de Comunicacion

Los protocolos de comunicación son reglas y convenciones que rigen el intercambio de información entre dispositivos en una red de computadoras. Establecen pautas para la transmisión de datos, el control de errores, la señalización, la autenticación y la privacidad de la información.

DNS

El Sistema de Nombres de Dominio (DNS) es un sistema que asigna nombres de dominio legibles por humanos a direcciones IP numéricas únicas de dispositivos y servidores en Internet. Esto permite a los usuarios acceder a recursos en Internet utilizando nombres de dominio fácilmente reconocibles en lugar de recordar direcciones IP numéricas complejas.

Enrrutamiento (Estatico y Dinamico)

Enrutamiento Estático: Se refiere a un método en el que un administrador de red configura manualmente las rutas en un enrutador. Las rutas no cambian automáticamente y deben actualizarse manualmente si hay cambios en la red.
Enrutamiento Dinámico: Se refiere a un método en el que los enrutadores intercambian información de enrutamiento automáticamente utilizando protocolos de enrutamiento. Los enrutadores pueden aprender sobre las redes vecinas y determinar las mejores rutas para enviar paquetes de datos en tiempo real.

DHCP

El Protocolo de Configuración Dinámica de Host (DHCP) es un protocolo de red que se utiliza para asignar automáticamente direcciones IP y otra información de configuración de red a dispositivos en una red. Permite a los dispositivos conectarse a una red y obtener automáticamente una dirección IP única sin necesidad de configuración manual.

NAT/PAT

NAT: Es un proceso que permite a múltiples dispositivos en una red privada compartir una única dirección IP pública. Traduce las direcciones IP privadas de los dispositivos internos en una dirección IP pública cuando se comunican con recursos externos en Internet.
PAT: Es una extensión de NAT que permite a múltiples dispositivos en una red privada compartir una única dirección IP pública mediante el uso de diferentes números de puerto. Asigna diferentes puertos a cada dispositivo interno para identificarlos de manera única en la red externa.
